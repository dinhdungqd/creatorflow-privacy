VEO – Google Flow
Site: labs.google/fx/tools/flow  ·  5 Modes  ·  Veo 3.1 Fast Veo 2
🚀 Getting Started ▼
1
Open labs.google/fx/tools/flow and navigate to a Flow project page.
2
Click the extension icon in Chrome's toolbar and pin it for easy access.
3
Open the Control tab in the side panel and select your desired mode.
4
Configure settings in the Settings tab before your first run.
⚠ Important

The extension only works when you are on a Flow project page. If the panel shows "not active", navigate to a project and refresh.

🎛️ Modes Overview ▼
Mode 1
Text-to-Video

Generate videos from text prompts. Separate prompts with blank lines.

Mode 2
Image-to-Video

Animate uploaded images with text prompts. 1–2 images per prompt.

Mode 3
Components-to-Video

Use character/component images with auto filename matching. Up to 3 images per prompt.

Mode 4
Text-to-Image

Generate images from descriptions. Supports 4K/2K/1K download.

Mode 5
Image-to-Image

Transform source images with prompts. Auto filename matching. Up to 10 images per prompt.

📋 Mode Details ▼

Mode 1 — Text-to-Video

  • Enter prompts in the text area, separated by blank lines, or upload a .txt file.
  • Configure duration: 8s or Concat.
  • Click Run to start batch generation.

Mode 2 — Image-to-Video

  • Upload images (supports multiple at once via file picker or drag-drop).
  • Enter one prompt per image (or per image pair).
  • Sort images by name or date, or reorder via drag-and-drop.
  • Configure 1–2 images per prompt in Settings.

Mode 3 — Components-to-Video

  • Upload component or character images.
  • Enter prompts describing the scene.
  • Enable "Auto-add character images" — the tool matches images to prompts by filename. For example, dog.png and cat.png are automatically attached when the prompt mentions "dog" and "cat".
  • Up to 3 images per prompt.
💡 Tip

Name your images clearly and descriptively (e.g. hero.png, dragon.jpg) so auto-matching works reliably.

Mode 4 — Text-to-Image

  • Enter image descriptions, one per blank-line-separated block.
  • Configure aspect ratio and image model in Settings.
  • Supports 4K / 2K / 1K auto-download.

Mode 5 — Image-to-Image

  • Upload source images and enter transformation prompts.
  • Enable "Auto-add character images" for filename-based matching.
  • Up to 10 images per prompt.
  • Supports 4K / 2K / 1K auto-download.

💡 Tips & Best Practices ▼
  • Use English prompts for best generation quality.
  • Separate prompts with a blank line — each block is one prompt.
  • Set ~3 concurrent prompts when Flow is stable; drop to 2 during peak hours.
  • Set a ~30 s delay between prompts to avoid rate limiting.
  • Name images clearly for auto-matching: dog.png, hero.jpg, etc. Supported formats: PNG, JPG, GIF (under 10 MB).
  • Lower concurrency = more stable runs with fewer errors.
  • Use Concat duration to chain multiple clips into one long video.
🔧 Troubleshooting ▼
❓ Extension not working / panel shows "not active"
You must be on a Flow project page at labs.google/fx/tools/flow. Navigate to a project and refresh the page.
❓ Cannot create video
The tool auto-retries every 30 s. This is normal — just wait and don't close the panel.
❓ Videos not downloading automatically
Go to Chrome Settings → Downloads and turn OFF "Ask where to save each file before downloading".
❓ Policy Error on a prompt
The prompt or image violates Google's content policy. The tool automatically skips that prompt and continues with the rest.
❓ Screen zoomed out unexpectedly
This is an automatic feature of the extension. Do not manually adjust zoom while a run is in progress.

Grok – xAI
Site: grok.com/imagine  ·  5 Modes  ·  Grok 3.1 Fast Grok 2
🚀 Getting Started ▼
1
Open grok.com/imagine and make sure you are logged in to Grok.
2
Click the extension icon and pin it to the toolbar.
3
In the Control tab, configure: Concurrent Prompts, Prompt Delay, Save Folder, and Auto Change File Name.
4
Select your mode and click Run.
⚠ Important

The extension only activates on grok.com/imagine. If the panel shows "not active", navigate to that URL and refresh.

🎛️ Modes Overview ▼
Mode 1
Text-to-Video

Generate videos from text prompts separated by blank lines.

Mode 2
Frame-to-Video

Animate images as start frame, or start + end frame pair.

Mode 3
Ingredients to Video

Character images auto-matched by filename. Up to 10 images per prompt.

Mode 4
Text-to-Image

Generate images from descriptions with configurable aspect ratio.

Mode 5
Image-to-Image

Transform source images with prompts. Auto filename matching.

📋 Mode Details ▼

Mode 1 — Text-to-Video

  • Enter prompts separated by blank lines, or upload a .txt file.
  • Click Run to start batch generation.

Mode 2 — Frame-to-Video

  • Upload or drag-drop source images.
  • Enter prompts for each image.
  • In Settings → Image Processing Option: choose Start frame only or Start + End frame.
  • Make sure you have enough images for the number of prompts.

Mode 3 — Ingredients to Video

  • Upload component or character images.
  • Enter prompts describing the scene.
  • Enable "Auto-add character images" for automatic filename-based matching.
  • Settings → Max Input Images per Prompt: 1–10.
💡 Tip

Name character images descriptively: hero.png, villain.jpg, dragon.png. The tool matches image filenames to words in your prompts.

Mode 4 — Text-to-Image

  • Enter image descriptions, one per blank-line-separated block.
  • Configure Aspect Ratio and Image Model in Settings.

Mode 5 — Image-to-Image

  • Upload source images and enter transformation prompts.
  • Enable "Auto-add character images" for filename-based matching.
  • Settings → Max Input Images per Prompt: 1–10.

Video Chaining with Extend

💡 Pro Tip

Use 6s extend or 10s extend as the video option for all prompts except the last one. This chains them into one continuous long video. Set the last prompt to a non-extend option.

💡 Tips & Best Practices ▼
  • Increase Random Delay if you hit rate limits frequently.
  • Start with 1 concurrent prompt and increase slowly to find your stable limit.
  • Use 6s extend / 10s extend to chain prompts into one long video — set the last prompt to a non-extend option.
  • Use the Quick Fix button if generation gets stuck.
  • Name character images descriptively for auto-matching: hero.png, villain.jpg.
  • For Frame-to-Video, ensure you have at least as many images as prompts.
🔧 Troubleshooting ▼
❓ Extension not active

❓ Downloads not working

❓ Login required error
You must be logged into Grok.com before using the extension.
❓ Not enough images for Frame-to-Video
Frame-to-Video requires at least one image per prompt. Upload enough images to match your prompt count.

Seedance – Dreamina / Jimeng
Sites: dreamina.capcut.com  ·  jimeng.jianying.com  ·  5 Modes  ·  Up to 2K 60FPS
🚀 Getting Started ▼
1
Open dreamina.capcut.com (or jimeng.jianying.com) and log in.
2
Click the extension icon and pin it to the toolbar.
3
Open the Control tab and configure your settings.
4
Select your mode and click Run.
⚠ Important

The extension only activates on dreamina.capcut.com or jimeng.jianying.com. If the panel shows "not active", navigate to one of those sites and refresh.

🎛️ Modes Overview ▼
Mode 1
Text-to-Video

Generate videos from text prompts. Blank line separator.

Mode 2
Frame-to-Video

Animate images as start frame or start + end frame pair.

Mode 3
Ingredients to Video

Character images auto-matched by filename. Up to 10 images per prompt.

Mode 4
Text-to-Image

Generate images from descriptions. Expanded aspect ratio options.

Mode 5
Image-to-Image

Transform source images. Supports image chaining in Edit Image mode.

📋 Mode Details ▼

Mode 1 — Text-to-Video

  • Enter prompts separated by blank lines, or upload a .txt file.
  • Configure duration: 4s–15s or Concat variants.
  • Click Run to start batch generation.

Mode 2 — Frame-to-Video

  • Upload or drag-drop source images.
  • Enter prompts for each image.
  • Settings → Image Processing Option: Start frame only or Start + End frame.

Mode 3 — Ingredients to Video

  • Upload component or character images.
  • Enter prompts describing the scene.
  • Enable "Auto-add character images" for filename-based matching.
  • Settings → Max Input Images per Prompt: 1–10.

Mode 4 — Text-to-Image

  • Enter image descriptions, one per blank-line-separated block.
  • Configure Aspect Ratio (includes 4:3 and 3:4) and Image Model in Settings.

Mode 5 — Image-to-Image (with Image Chaining)

  • Upload source images and enter transformation prompts.
  • Enable "Auto-add character images" for filename-based matching.
  • Enable Edit Image mode to activate Image Chaining: each prompt's output image automatically becomes the input for the next prompt.
💡 Image Chaining

Image Chaining is a Seedance-exclusive feature. Enable Edit Image mode in Settings to chain prompts — great for iterative style transformations or storytelling sequences.

Video Chaining with Concat

💡 Pro Tip

Use Concat duration to chain multiple video clips into one long video. Each prompt generates a clip that is appended to the previous one.

  • Configure Aspect Ratio in Settings.

Mode 4 — Image-to-Image
