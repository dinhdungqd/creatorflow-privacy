Privacy Policy

CreatorFlow – Tools for Grok, VEO & AI Creators  |  Last updated: May 2026

CreatorFlow does not collect, store, or transmit any personal data to external servers. All processing happens locally in your browser.

1. What We Collect

CreatorFlow does not collect any personally identifiable information. The extension operates entirely within your browser and does not send any user data to third-party servers.

2. Local Storage Only

The extension uses Chrome's built-in chrome.storage.local API solely to remember your preferences between sessions, such as:

  • Last active tab (VEO, Grok, Seedance, Meta)
  • Download folder name and file prefix settings
  • Automation configuration options you set

This data never leaves your device and is never transmitted anywhere.

3. Permissions Explained

  • storage – Save your local preferences only
  • tabs – Detect which AI site is active to auto-switch the panel; reload tabs after install
  • activeTab – Interact with the currently open AI site tab
  • downloads – Save generated videos and images to your local device
  • sidePanel – Display the CreatorFlow panel in Chrome's side panel
  • debugger – Simulate precise mouse click events on AI site buttons that block standard DOM events. No page content is read or recorded.

4. Host Permissions

The extension only activates on these specific sites:

  • labs.google (Google Flow / VEO)
  • grok.com (Grok / xAI)
  • dreamina.capcut.com, jimeng.jianying.com (Seedance)
  • meta.ai (Meta AI)

The extension does not run on any other websites.

5. No Remote Code

All extension logic, UI, and configuration data is bundled locally within the extension package. No code or configuration is fetched from external servers at runtime.

6. No Analytics or Tracking

CreatorFlow contains no analytics, telemetry, crash reporting, or any form of usage tracking. We have no visibility into how or when you use the extension.

7. No Third-Party Data Sharing

We do not share, sell, or disclose any data to third parties — because we do not collect any data in the first place.

8. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect information from children.

9. Changes to This Policy

If this policy is updated, the new version will be included in the extension update published to the Chrome Web Store. The "Last updated" date at the top will reflect any changes.
